# HomePage
User can see all the movie and tv show related details in the home page like top movies/tvShows, popular movies/tvShows and also top rated movies/tvShows.

# Navbar
in navbar user can see genre, year and search features using genre dropdown user can select any particular genre that user whishes and see all the tvShows/movies related to that particular genre.
same goes for year user can select any particular year and see all the tvShows/movies released in that year.

# Movie/TV Show details
user can click on any movie/tvShow card and navigate to the details page for that specific movie/tvShow and see the overview, cast and crew, popularity, IMDb rating and user provided rating which genre it belongs to and also year of releasing.

# User Rating
user can provide the rating for that specific movie/tvShow by selecting the Stars on the stars each star represents 2 rating, example if user selects 4 stars then rating display would be 8.
